# 微光集 (Lightlet) — 产品需求提示词

---

## 一、产品概述

**微光集**是一个为 INFP 性格设计的每日记录小程序。它不是一个传统的待办清单（Todo List），而是一个**生活痕迹收集器**。

### 核心理念

- **记录做过什么，而不是追着做什么** — 传统待办工具用"未完成"制造焦虑，微光集用"已完成"制造成就感
- **用隐喻替代指令** — 完成一件事 = 点亮一颗星，而不是打一个勾
- **用积累替代消除** — 你做过的每件事都在星空中留下痕迹，越积越多，而不是做完就消失
- **不催、不罚、不制造负罪感** — 未完成的事不会变红、不会弹出提醒，只是安静地等待

### 参考风格

类似 **Focus Traveler** app 的美术风格——温暖、治愈、有质感，不是冷冰冰的效率工具。

### 目标用户心理

INFP 容易在「我要做什么」上卡住，但「我做了什么」是可以轻松说出来的。微光集不逼你做计划、不催你打勾，只是温柔地记录。回头看时自然形成一种「原来我做了这么多」的认知——这正是 INFP 需要的正反馈。

---

## 二、核心概念：四个空间

微光集有两套视觉模式，每套模式有两种状态，共四个空间：

| 空间 | 模式 | 数据范围 | 视觉 | 用途 |
|---|---|---|---|---|
| **今日星空** | 星夜模式 | 当天 pending + done | 深色背景 + 琥珀金光点 | 日常记录、收集成就感 |
| **遗忘之海** | 星夜模式 | expired（超期未完成） | 深色背景 + 暗淡的星 | 回看被遗忘的事 |
| **今日脉络** | 脉络模式 | 当天 done | 白底 + 黑色圆点 + 连线 | 查看完成事项的关联 |
| **档案室** | 脉络模式 | expired（超期未完成） | 白底 + 灰点 + 虚线 | 未完成事项的关联网络 |

关键理解：**遗忘之海和档案室是同一份 expired 数据的两种视图**——不是两种不同的状态。用户在星夜模式看到的是"遗忘之海"，切到脉络模式看到的是"档案室"。不需要用户逐个选择"遗忘"还是"归档"。

### 空间流转

```
创建事项(pending) → 今日星空
完成事项 → status=done，停留在完成的星空
未完成 + 超过一段时间 → status=expired，沉入遗忘之海/档案室
可归档(从今日移除但保留在记录里) / 可删除(永久移除)
```
