# 2026-07-15

## 微光集 — 每日记录小程序 MVP

- 家辉想要一个适合 INFP 性格的每日记录工具，参考 Focus Traveler 美术风格
- 核心洞察：传统待办清单是 INFP 的敌人——固定任务+打勾=压力，"没打勾"=负罪感
- 设计方向：**记录做过什么，而非追着做什么**。隐喻：每件小事是一点微光，汇集起来是星空
- 创建了完整方案 + 可交互 MVP 原型：`微光集/index.html`
  - Canvas 星空画布 + 粒子动画
  - 自由输入，点击"点亮"完成任务（非打勾）
  - localStorage 持久化
  - 时间自适应问候语
  - 包含方案规划标签页（设计原则、与传统的对比、开发路线图）
- 技术：纯前端单文件 HTML，深色主题 + 森林绿点缀
- 家辉确认方向，补充需求：脉络模式（白底黑点+连线）、30天星空缩放、AI关联建议
- 制作美术概念设计文档：art-direction.html（双模式视觉预览、色彩系统、动效语言、设计原则）
- 推送到 GitHub：https://github.com/elsewherrre/lightlet
